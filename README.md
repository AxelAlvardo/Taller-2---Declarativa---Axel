## Descripción
Este proyecto contiene la resolución completa de los **12 ejercicios** del *Taller 02* de la asignatura **Programación Declarativa**, implementados en el lenguaje **Racket**.  
Cada ejercicio incluye su respectiva función, llamada de ejemplo y salida esperada en consola, siguiendo una sintaxis clara, estructurada y coherente.

---

## Contenido de los ejercicios

| Nº | Tema principal | Descripción breve |
|----|----------------|------------------|
| 1  | Listas y condicionales | Contar elementos positivos en una lista |
| 2  | Mapeo de listas | Generar una lista con los cuadrados de los números pares |
| 3  | Recursión | Calcular el factorial de un número |
| 4  | Map | Elevar cada elemento de una lista al cubo |
| 5  | Filtros y suma | Sumar los números impares de una lista |
| 6  | Predicados | Verificar si una lista contiene números negativos |
| 7  | Recursión y acumuladores | Generar una lista con sumas acumuladas |
| 8  | Cadenas | Concatenar dos cadenas de texto |
| 9  | Filtros y transformación | Obtener el doble de los números mayores que 5 |
| 10 | Listas | Invertir el orden de los elementos de una lista |
| 11 | Funciones de orden superior | Aplicar una función a todos los elementos de una lista |
| 12 | Promedio y filtros | Calcular el promedio de los números mayores a 5 |
